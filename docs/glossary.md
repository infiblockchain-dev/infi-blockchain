# INFI Glossary

## INFI

The blockchain network.

## Invertx

The native gas token of INFI.

## test InvertX

The INFI Testnet gas token. It is free testnet-only gas, non-tradable, and has no redeemable real-world value.

## Gas

The fee paid to execute transactions.

## EVM

Ethereum Virtual Machine. The execution environment for Solidity smart contracts.

## RPC

Remote Procedure Call. Wallets and dapps use RPC to read chain data and submit transactions.

## Validator

A node that participates in consensus and finality.

## Full Node

A node that verifies chain data but does not necessarily produce blocks.

## Archive Node

A node that stores full historical state.

## INFI Scan

The official explorer for INFI.

## WalletConnect

A protocol and app SDK ecosystem for connecting wallets and dapps, including QR and mobile linking flows.

## SbSe Protocol

Future governance logic engine node for INFI.

## Genesis

The first state of the blockchain.

## Finality

The point where a block is considered irreversible under the consensus rules.

## Bridge

A system that transfers assets or messages between chains.

## Tokenized Asset

An on-chain representation of digital or real-world value.

## INFI Domain

An on-chain `.infi` name that can resolve to wallets, content, dapps, assets, or service records.

## VPN Provider Node

A future INFI network node that provides decentralized VPN relay service outside validator consensus.
